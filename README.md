# Hotel Booking Analysis and Cancellations Insights 

## 📌 Project Overview

This project focuses on analyzing hotel booking data to understand customer behavior, booking trends, and cancellation patterns. The analysis helps in identifying key factors that influence booking cancellations and provides insights that can support better decision-making in the hotel industry.

---

##  Objectives

* To analyze hotel booking patterns
* To identify factors affecting reservation cancellations
* To understand seasonal trends in bookings
* To provide insights for pricing and promotional strategies

---

## Key Questions Addressed

1. What factors influence hotel reservation cancellations?
2. How can hotel reservation cancellations be reduced or managed effectively?
3. How can data analysis help in improving pricing and promotional strategies?

---

## 🛠️ Tools and Technologies Used

* **Python** – for data analysis
* **Pandas** – for data manipulation and cleaning
* **NumPy** – for numerical operations
* **Matplotlib** – for data visualization
* **Jupyter Notebook** – for interactive analysis
* **Kaggle** - for data source
---

## Exploratory Data Analysis (EDA)

In this project, various visualizations were created to understand:

* Booking distribution across hotel types
* Cancellation trends
* Monthly booking patterns
* Average Daily Rate (ADR) trends
* Customer segments and market sources

---

## Some Key Insights

* City Hotels have higher bookings but also higher cancellation rates
* A significant portion of bookings are canceled
* Longer lead time increases chances of cancellation
* Bookings show seasonal trends (peak in certain months)
* Online travel agents contribute the most bookings and cancellations

---

## 💡 Recommendations

* Improve cancellation policies to reduce losses
* Use dynamic pricing strategies based on demand
* Focus on high-cancellation segments like Online TA
* Plan resources according to seasonal trends
